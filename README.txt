Send POST requests http://192.168.0.69/input

the following commands are applicable:
forward - move the robot forward roughly 8 inches
back - move the robot backwards roughly 8 inches
left - move the robot left and then forward 8 inches
right -	move the robot right and then forward 8 inches
auto - set the robot into wallfollowing mode
manual - if the robot is in wallfollowing mode, this will switch it back to manual mode.
